# Advanced Calculator

A professional-grade command-line calculator demonstrating object-oriented programming, design patterns, comprehensive testing, and continuous integration.

## Overview

This calculator application showcases software engineering best practices including the Factory pattern, Singleton pattern, calculation history management, and 100% test coverage. The project emphasizes clean architecture with clear separation between operations, business logic, and user interface.

## Features

**Core Operations**
- Addition, Subtraction, Multiplication, Division
- Comprehensive error handling including division by zero
- Support for both integers and floating-point numbers

**Calculation Management**
- Factory pattern for creating calculation instances
- Complete calculation history tracking
- Singleton pattern for history management

**Interactive Interface**
- REPL (Read-Eval-Print Loop) for continuous interaction
- Special commands: help, history, clear, exit
- Input validation with helpful error messages

**Software Engineering**
- 100% test coverage with pytest
- Parameterized tests for efficient scenario coverage
- Both LBYL and EAFP error handling paradigms
- Automated CI/CD with GitHub Actions
- Comprehensive documentation and type hints
